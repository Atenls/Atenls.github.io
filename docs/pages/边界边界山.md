# 边界边界山

> 本区域为 `边界山` 的内部秘境，你需要通过边界山传送门进入此地。<br>
本区域实际处于 `外海` 中。

---

## Intro

你可以在泠风峡谷底部的聚集地中购买流转空间钟，并于 **边界门** 处使用以前往此地。<br>
或通过 `界域引继者` 对着 **边界门** 使用，亦可前往此地，但 `界域引继者` 将进入一次 120 mins 的长冷却。

## Description

本区域由双峡谷构成，经一个山洞连通。<br>
当你进入时，你将在 `(x:766,y:107,z:-1359)` 处，你也可以从这里返回边界山。<br>
当你死亡时，你将于此处复活。但你离线后，将不会再回到此地<br>

本区域存在通关概念，在你击杀了一定量的暗能咒术师后，你可以于双峡谷的末端 **消耗体力** 离开，并领取通关奖励。<br>
奖励将受你已击杀的暗能咒术师、暗影标记状态影响。

`Lv.265` ~ `Lv.285`

## 环境影响

外海的光照是可受环境定义的，边界边界山属暗元素区域，在此区域，暗属性将受到增益。<br>
部分伤害将视乎 **你所处位置的方块亮度** 而有所变化。<br>

### 对玩家伤害

#### 暗咒

暗咒将在黑暗环境中造成更高的伤害：<br>
当你的环境亮度低于 **7** 时，每降低 **1** 点亮度，都将对伤害额外增益 **75%** ，即 $Damage = Damage * (1+ \frac{3}{4} * [Max(7 - BlockLight , 0)])$ <br>
因此在任何情况下，都请尽可能保持行走在道路 (道路通常有 6 ~ 8 亮度) 上。

参考:

```TEXT
Case 1:
假定你的暗属性抗性为 0，同时，行走在亮度为 0 的区域，
你受到一次来自 暗能咒术师 (Elite) 的暗咒攻击，将受到 867895 * (1+Max(0,(26875-0)/1000)*2) * (1 + 5.25) = 296.98M 伤害

Case 2:
假定你的暗属性抗性为 25000，同时，行走在亮度为 8 的区域，
你受到一次来自 暗能咒术师 (Elite) 的暗咒攻击，将受到 867895 * (1 + (26875-25000)/1000 * 2) * (1 + 0) = 4.12M 伤害

Case 3:
假定你的暗属性抗性为 26875+，同时，行走在亮度为 8 的区域，
你受到一次来自 暗能咒术师 (Elite) 的暗咒攻击，将受到 867895 * (1 + 0) * (1 + 0) = 867.9K 伤害
```

#### 暗能锁链

暗能锁链会在移动过程中吸收路径中的能量：<br>
当你们的距离超过 **8** 格，每提升 **2** 格，都将对暗能锁链提供 **100%** 的增益，即  $Damage = Damage * (1 + {\large Max[Floor({\Large \frac{Distance - 8}{2}}) , 0]})$ <br>
因此请尽可能近距离击杀暗能咒术师。<br>

> 距离：欧几里得距离
