## portal 指令

输入指令：`/portal` 后，你可以选择你要传送的地点。

当你已接取任务，如任务目标支持 Warp，可以使用 `/portal warp` 以快速抵达目的地。

 或你可以使用指令：`/portal 代号` 进行传送

> 部分代号展示：
> 1c（傲林镇）, 10b（鬼骑士）, h60（晨曦矿洞）, d72（灵魂峡谷）, quest3（空间站）, rainmaker（唤雨台）

除此之外，代号还支持怪物名字，部分素材掉落以及珍惜产物，如:
皓牛，蘑菇肉，深井水等。

**温馨提示：此指令需要拥有 c2 以上月卡（Noctambulist）才可以使用以上指令。**

## dptest 指令

- `/dptest bypasscraft-散-<3/4/5>` - 制作散 3 4 5（需要开启 qs）
    - 散3（75级）：散\*6 成就勋章\*4 mope\*2 dpe\*32400
    - 散4（90级）：散\*36 成就勋章\*16 mope\*8 dpe\*128000
    - 散5（120级）：散\*216 成就勋章\*128 mope\*32 dpe\*1080000
- `/dptest valiantsell-<光晦/武器>` - （开启/关闭丢弃分解）（Mos.Top 或以上特权）
- `/dptest bossrollbc` - 以开关自身*所有已支持的 Boss* 的出货公告。（目前仅支持 40b ~ 120b）
- `/dptest drop-sell` - 开启/关闭丢弃未知物品快速分解（Mos.Top 或以上特权）
- `/dptest pickaxerefresh` - 刷新 bug 的 13 速黑曜石镐（2024/1/15 前制作的）
- `/dptest d48-osmode` - 切换至源石模式。舍弃其他所有产出，仅产出高额源石。
- `/dptest d48-select-<num>` - 固定设置开启数量。可供选择的数量为 4 / 16 / 64 / 256 / 2048。
    *锁上限不锁下限，例锁 2048 时，包内不足 2048 仍可以开启*
- `/dptest d48-fil` - 切换至丢弃杂物模式。开启后将自动丢弃强化券、延时卡、史前手札、档案页、史前戒指、补给箱碎片。
- `/dptest messageblock` - 部分数字消息提醒

## 随机抽取指令

- 使用指令 `/roll` 以随机 roll 点 (1500 dPe)
- 使用指令 `/roll p` 以随机抽取一个玩家 (3000 dPe)
- 使用指令 `/roll bg` 以购买 20 Mins. 的全局免费 (500k dPe)
- `/roll ip` - 随机以 ip 抽选人，适用下方 ignore 池
- `/roll p bp` - 随机抽取人并显示该玩家当前赛季 bp 点数
- `/roll p level` - 随机抽取人并显示该玩家当前等级
- `/roll ignore [Player1] <Player2>`
- `/roll ignore clear` - 清除屏蔽列表。

**附注**：通过 ignore 进行屏蔽玩家(组)是覆写操作，每次新执行都会完全覆盖上次设定。如：
  - `/roll ignore test001 test002` → 此时屏蔽了 test001、test002，
    - 而后再次输入 `/roll ignore test003` → 此时只屏蔽了 test003

- `/idtas` - 开启后可以自动出售你背包中的未鉴定物品并合成源石残片!
>该功能为 Metaphysician 或以上月卡等级专属功能!

---

## mobas 指令

- `/mobas 怪物名称` - 兑换技艺书
>该功能仅对C4开放
---

## msglog 指令

- `/msglog` → 直接查询最后一页聊天记录（倒序）
- `/msglog [page]` → 查询某页的聊天记录
- `/msglog [player]` → 查询某人的聊天记录
- `/msglog [player] [page]` → 查询某人的聊天记录的第某页
- `/msglog ai` - 查看王老吉的信息

## refresh（刷新）指令

- `/refresh` - 刷新物品

---

## 常见快捷输入

- `%+数字` - 展示对应背包快捷栏道具
- 每日 1（遗落之地）每日 2（时光回廊）每日 3（灵魂峡谷）
- 主城与领主怪物需要再后面加 "/"
    - 如：`2c/`（丘山城） `10b/`（鬼骑士）
- `1dN` - 从范围内随机一个整数（N 至多为 999）

## 调试设置

- `F3 + A` = 重新加载区块
- `F3 + B` = 显示碰撞体积
- `F3 + D` = 清空聊天记录
- `F3 + F` = 在渲染距离间循环(按 shift 来反向循环)
- `F3 + G` = 显示区块边界
- `F3 + H` = 显示高级提示框
- `F3 + N` = 在创造和旁观模式间切换
- `F3 + P` = 失去焦点时暂停
- `F3 + Q` = 显示此列表
- `F3 + T` = 重新加载资源包
