# 制式符文

## 简介

制式符文是一种通过 `Kc` 购买的符文，拥有固定的搭配，可以提供大量额外的属性。<br>
制式符文现分为 `核心符文` `附属符文`，两类符文均可独立购买、使用。<br>
当你为 `核心符文` 购置「附属槽位」时，可将已拥有的 `附属符文` 镶嵌于其上，节约符文位。「附属槽位」至多拥有 `3` 个。

## 符文明细

所附价格均为 `Kc`<br>
属性中所提到的非百分比或显然为固定数值的成长性属性，单位均为基准属性单位。(你可通过 /cal baseattr 查询)
