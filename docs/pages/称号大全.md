# DreamPainterIV（DPIV）服务器称号大全

由于部分内容未能得到详细数据，因此无法将所有服务器称号均于本文档完全介绍，若有疏漏，请及时联系玩家 **Miyazono_nana** 进行完善补充。
你可以使用指令:`/prefix help`获取关于称号的帮助。

**除特别标注外，称号仅展示玩家个性或是游戏内各环节的探索深度，并不会带来任何加成。**

### **限定称号（月卡、内测、指导员认证、服务器比赛）**

- 月卡：月卡/季卡生效期间可以更换
    - 指令：`/prefix monthlycard` 或 `/prefix mc`
- 战令：战令生效期间可以更换
    - 指令：`/prefix luxurypass` 或 `/prefix premiumpass`
- 内测称号：开拓者、DP合作社等
- 指导员认证：御坂****
    - 指令：`/prefix misaka`
- 比赛奖励：Tempest Rower、Excited、天若有情天亦老等。
- 八周年签到限定：携游

### **专属称号（自定义称号）**

- `/changeprefix` - 修改自定义称号（冷却时间一个月）
- `/prefix custom` - 展示自定义称号
- 赞助 1000 CNY 解锁，可以自定义颜色、字符、长度（3-15）。
