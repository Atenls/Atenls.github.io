# 制式符文

## 简介

制式符文拥有固定的属性搭配，可通过 `Kc` 购买。 1 CNY = **100** Kc。

## 符文明细

所附价格单位均为 `Kc`。百分比属性以 `%` 标注；固定属性直接显示数值。<br>
配置中通过公式乘以等级基准值的属性以 `unit` 表示，例如 `0.25 unit` 表示对应属性等级基准值的 `0.25` 倍，不代表固定面板数值。
