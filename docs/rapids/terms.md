1. “本服务器”指 Rapids 及其附属或关联之游戏服务、网页服务、社群服务及测试环境。

3. “玩家”指任何访问、注册、连接、游玩、观看、测试、参与活动或使用相关服务之自然人。
4. “帐户”指玩家用于连接本服务器及相关平台之身份凭证、角色、绑定资料及对应权限。
5. “游戏资产”指角色资料、等级、货币、装备、仓库、地产、道具、任务进度、排行榜成绩、活动记录及其他虚拟资料。
